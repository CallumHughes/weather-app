The Weather App API is a small REST API behind the weather-app web client. It
returns current weather for a free-text location and manages the signed-in
user's search history and favourite locations. Responses are JSON; errors
always use the shared [error envelope](/docs/errors).

## Base URLs

| Environment | Base URL | Notes |
| --- | --- | --- |
| Production | same origin as the web app, under `/api/*` | The Next.js BFF reverse-proxies to the Fastify API, so session cookies stay first-party |
| Local development | `http://localhost:3000` | Fastify directly (`pnpm nx dev server`) |

All documented endpoints are versioned under `/api/v1` (except `/health`,
which is unversioned for infrastructure probes).
